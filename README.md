# A modern, multi-city weather application built with SwiftUI, featuring real-time weather data, hourly forecasts, and a nice, intuitive interface.

![Simulator Screen Recording - iPhone 17 Pro Max - 2025-12-12 at 00 38 10](https://github.com/user-attachments/assets/afb6f64f-dd8a-476e-be97-a069cad2265a)

### *Features:*

**Implemented**

* Real-time Weather Data from Open-Meteo API
* Multi-City Support with tab navigation
* Hourly Forecast (next 12 hours)
* 10-Day Daily Forecast
* Day/Night Mode with automatic detection
* Pull-to-Refresh functionality
* Error Handling with fallback to mock data
* Nice UI with gradient backgrounds, SF Symbols and glass effect implemented

**Under Development**

* Location-based weather detection
* Custom search for new cities
* Weather alerts and notifications
* Air quality index display
* Weather widget for Home Screen
* Additional weather details (UV index, visibility, etc.)
* Unit conversion (Celsius/Fahrenheit)

**Key Technologies**

* SwiftUI for declarative UI
* Combine for reactive programming
* Async/Await for network calls
* Codable for JSON parsing
* SF Symbols for consistent icons

*Uses Open-Meteo for free weather API - This app is still under development. Features may change*
